Parameter-Efficient Fine-Tuning of Linear Attention Models

Columbia University New York, NY, USA

  • Developed a systematic training framework to investigate how different LoRA* target modules influence fine-tuning performance in linear-attention-based language models (trained on RetNet, Gated Linear Attention, and DeltaNet).

Sep 2025 - Jan 2026

VideoBrain

University of California, Irvine Irvine, CA

  • Proposed VideoBrain, an end-to-end framework enabling VLMs to adaptively sample video frames through dual complementary agents: CLIP-based agent for semantic retrieval and Uniform agent for dense temporal sampling.
  • Designed a behavior-aware reward function with a data classification pipeline to prevent reward hacking and reduce unnecessary agent invocations.
  • Implemented GRPO multi-turn reinforcement learning framework for Qwen3-VL-8B. Built a Ray-based caching to address video decoding memory bottleneck during large-scale sampling, reducing CPU memory usage by 40%.
  • Achieved +3.5% to +9.0% accuracy improvements on 4 long video benchmarks while using 30-40% fewer frames. Strong generalization to short videos with +2.4% to +5.1% accuracy using 46-47% fewer frames.
Jun 2024 – Nov 2024

Self-Correcting Human Motion Synthesis with Video Analysis

University of California, Irvine Irvine, CA, USA

  • Presented a multimodal framework for human-motion generation, editing, and comprehension.
  • Introduced the Motion Alignment Score (MAS), a motion–text alignment metric that renders motion as video, encodes it, and computes similarity scores between video and text embeddings.
  • Implemented a self-correcting generation loop: the Task Planner provides context enhancement via multimodal LLM reasoning; the Motion Generator handles synthesis and precise local edits; the Motion Reviewer (an instruction-tuned VLM) performs autonomous quality evaluation. MAS improved from 28.61 (first round) to 29.40 after editing.
